#  Smart Study Focus Analyzer
An AI-based system that predicts student focus level and provides insights, reasons, and suggestions.

##  Overview

This project is a simple AI-based system that predicts a student's focus level based on daily habits such as sleep, screen time, study hours, and breaks.

It not only predicts focus but also provides:

* Reasons behind the prediction
* Personalized suggestions
* Productivity score and performance level

---

##  Features

* Focus Level Prediction (High / Medium / Low)
* Explainable AI (Reasons for prediction)
* Smart Suggestions for improvement
* Productivity Score (0–100)
* Performance Category (Excellent, Good, Average, Poor)

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn (Decision Tree Classifier)

---

## ▶️ How to Run

1. Install required libraries:

```
pip install pandas scikit-learn
```

2. Run the program:

```
python main.py
```
3. Enter input values when prompted.

##  Example Output

* Predicted Focus Level
* Reasons for low/high focus
* Suggestions to improve
* Productivity Score

##  Conclusion
This project demonstrates how machine learning can be used to analyze daily habits and provide meaningful insights for improving productivity.
